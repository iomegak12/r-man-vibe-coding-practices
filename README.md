# R-MAN E-Commerce Customer Management System

Microservices-based e-commerce customer management platform.

## Services

### Authentication Service (ATHS)
User authentication and authorization service.
- **Location**: [back-end/aths](back-end/aths)
- **Documentation**: [API Docs](back-end/aths/docs/API_DOCUMENTATION.md)
- **Swagger UI**: http://localhost:5001/api-docs

## Quick Start

```bash
# Authentication Service
cd back-end/aths
npm install
node server.js
```
